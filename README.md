NXRuby
2DゲームライブラリであるDXRubyのAPIを参考にSDL3で作成したゲームライブラリです。

## 状況
開発中

## 特徴(目標)
- mrubyでスクリプトを書ける
- SDL3によるモダンなレンダリング
- Windows/Linux/Macサポート
- WASM対応によるブラウザゲームサポート
(現在はWindowsでしかビルドできません)
